# ⚡ Go ElectriQ

Go ElectriQ is an Electric Online Cab Booking Website that allows users to easily book electric vehicles for eco-friendly travel.  
The platform focuses on sustainable transportation and provides a simple and user-friendly interface for booking rides.

---

## 🚀 Features

- User Registration & Login
- Book Electric Cabs Online
- View Available Vehicles
- Easy Ride Booking Interface
- Responsive Design for Mobile & Desktop
- Environment-Friendly Transportation Concept

---

## 🛠️ Tech Stack

Frontend:
- React.js
- Tailwind CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB
